# Testes

Esta pasta está reservada para testes automatizados em versões futuras do projeto.

Nesta entrega acadêmica, o foco foi a construção do fluxo completo de Ciência de Dados, incluindo coleta, limpeza, integração, análise exploratória, modelagem e interpretação dos resultados.

Em uma evolução futura, esta pasta poderá conter testes para validar:

- leitura dos arquivos;
- existência das colunas esperadas;
- ausência de valores ausentes;
- consistência do número de municípios;
- padronização dos nomes das variáveis;
- funcionamento das funções de tratamento;
- funcionamento dos modelos de classificação.

A pasta foi mantida para preservar uma estrutura compatível com boas práticas de projetos de Ciência de Dados.
