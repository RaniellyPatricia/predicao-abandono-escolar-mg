# Modelos

Esta pasta está reservada para armazenar modelos treinados em versões futuras do projeto.

Nesta entrega acadêmica, os modelos foram treinados, avaliados e comparados diretamente no notebook `04_modelagem.ipynb`, utilizando Google Colab e Scikit-learn.

O melhor modelo identificado foi o **Random Forest Balanceado**, selecionado por apresentar o melhor F1-score macro e melhor equilíbrio entre as classes de abandono escolar.

Em uma evolução futura do projeto, esta pasta poderá armazenar arquivos como:

- modelos treinados em formato `.pkl` ou `.joblib`;
- versões diferentes dos modelos;
- arquivos de configuração;
- métricas finais salvas;
- pipelines completos de pré-processamento e modelagem.

A pasta foi mantida para preservar uma estrutura compatível com boas práticas de projetos de Ciência de Dados.
